# CS307 Project 1

This project is for comparing using file IO and DBMS.

## Usage

* Build project
    ```shell script
    mvn clean install
    ```
* Load test dataset
    ```shell script
    java -jar project1-0.0.1-jar-with-dependencies.jar -load <file>
    ```
